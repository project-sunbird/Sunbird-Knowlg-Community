# Collection Editor - V1

Collection Editor which allows user to create a group of contents. It can be (ECML, Pdf, HTML, H5P, Epub, Youtube, Mp4/Webm)

![](<../../../.gitbook/assets/image (1) (1).png>)

### Tech Stack

* **Angular v9**
* **HTML, SCSS**
* **Typescript**
